This project is a slight extension of a paper which deals with the origin of monogamy(See reference1).
The baryplot_col in code is a collection of basic library for simulation in thesis.
Also final_project_fig3_modified(backup) and final_project_fig4 are code for simulating result in figure 3 and 4 in paper.

Reference
1.Schacht, R., Bell, A. The evolution of monogamy in response to partner scarcity. Sci Rep 6, 32472 (2016). https://doi.org/10.1038/srep32472
